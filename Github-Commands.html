Github commands

1.git clone:

Use to download existing source code from remote repository

Syn:

git clone < https://name-of-the-repository-link>

2.git branch:

Use to give chance all developers work together on same project

Syn:

git branch < branch-name> for local repo

git push -u < remote> < branch-name> for remote repo

git branch or git branch --list to view branch

git branch -d < branch-name> to delete branch

3.git checkout

To switch from one branch to another

Syn:

git checkout < name-of-your-branch >

4.git status

it use to give all information about current branch

Syn:

git status

5.git add

use to includes changes in our file into our next commit

Syn:

git add < file >

git add -A (to add everything at once)

6.git commit

commit is like saving some changes what we done every time with message

Syn:

git commit -m "commit message"

7.git push

whatever changes we made commited, to send that to the server the push is used

Syn:

git push < remote > < branch-name >

git push --set-upstream < remote > < name-of-your-branch > ,for new branch

git push -u origin < branch_name >

8.git pull

use to get updates from remote repository and its a combination of git fetch and git merge

Syn:

git pull < remote >

9.git revert

it is use to undo our commits and to see commit history we use

git log --online

and by which we get hashcode,,that use to revert commit

Syn:

git revert 3321844

10.git merge

used to merge with parent branch after completing all commits

Syn:

git merge < branch name >
